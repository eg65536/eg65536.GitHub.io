撒切尔继续对保守党思想产生催眠般的影响，自她被罢免以来开始的心理剧引发了一场尚未解决的内部危机。除了保守党政治，撒切尔引入的变革已经成为一种共识，直到现在才开始受到持续的挑战，因为经济停滞和社会衰退造成了一种永久危机感。 撒切尔并不以哲学著称，倾向于用实际术语讨论她的信念和政策。然而，她是唯一一个被经常描述为“-主义”的英国首相。即使在当时，人们对她的世界观是否连贯，还是充满矛盾也有很多争议。正如约翰·坎贝尔——撒切尔首部多卷传记的作者——所说： “撒切尔主义的核心悖论是，撒切尔夫人主持并庆祝了一种狂热的物质主义文化——‘乐趣、贪婪和金钱’——这与她自身的价值观根本冲突，她的价值观本质上是保守的、老派的和清教徒式的。” 然而对她来说，这并不是悖论——或者至少她没有打算让它成为悖论。撒切尔离任后不久，学者雪莉·罗宾·莱特温写了一本总体友好的书，名为《撒切尔主义的解剖》，这本书比任何其他东西都更好地解释了前首相的思想如何融合。这无疑得益于她的儿子奥利弗曾是撒切尔在唐宁街10号的政策团队中的一员。 莱特温认为，撒切尔主要感兴趣的是推广她所说的“活力美德”： “撒切尔主义偏好的个人，首先是一个简单的列表：正直、自给自足、充满活力、冒险、独立思考、忠诚于朋友、对敌人坚定。” 在撒切尔的心中，这些美德被降级，取而代之的是柔和的美德：谦逊、温柔、对共识和一致的渴望。这正是70年代英国似乎面临的危机的核心。正如她在1977年所说——“反对社会主义的真正理由不是其经济效率低下……更为根本的是其基本的不道德性。”她在次年的一次著名演讲中详细阐述了这一观点，题为《基督教与政治》。 “一旦你让人们认为所有这些事情都可以由国家来做，而且让私人去做反而是次等的甚至是贬低的行为……那么你就会开始剥夺人们作为人类的一个基本要素——个人道德责任。你实际上会让他们失去人类善良的乳汁。如果你允许人们把所有的个人责任都交给国家，那么终有一天——事实上这一天已经不远了——人们愿意为了人类的福祉提供的税收将被认为远远少于个人曾经愿意出于对邻居的爱而给予的东西。” 从她的角度来看，因此，将所有权从国家转移到个人和家庭的经济政策赋予人们自由，让他们能够创业地使用自己的财富，并减少了国家在分配劳动力和资源方面的作用。这具有基本的道德意图。目标不仅仅是一个更加富裕的英国，还要让更多的人培养那些充满活力的美德，以克服国家的道德堕落。正如莱特文所说： 或者如撒切尔在1981年对《星期日泰晤士报》所说的那样，“经济是方法，目标是改变心灵和灵魂”。因此，她执政期间的核心政策决策包括：出售公租房、以折扣价格向公众出售私有化公司的股份、将税收从收入转向消费等。房产对她来说尤为重要，是家庭生活的核心，跨越多个世代。除了购房权，她还引入了按揭利息扣除，甚至考虑过单独的按揭利率。这也是不成功的人头税背后的思想基础，旨在保护“拥有大房子的小老太太”免受税率影响。 一旦官府认真发起对妖术的清剿，普通人就有了很好的机会来清算宿怨或谋取私利。这是扔在大街上的上了膛的武器，每个人——无论恶棍或良善——都可以取而用之。在这个权力对普通民众来说向来稀缺的社会里，以“叫魂”罪名来恶意中伤他人成了普通人的一种突然可得的权力。对任何受到横暴的族人或贪婪的债主逼迫的人来说，这一权力为他们提供了某种解脱：对害怕受到迫害的人，它提供了一块盾牌；对想得到好处的人，它提供了奖赏；对妒嫉者，它是一种补偿；对恶棍，它是一种力量；对虐待狂，它则是一种乐趣。 我们在这里所瞥见的，是否是一个已被人口过度增长、人均资源比例恶化、社会道德堕落所困扰的社会所遭受到的一种道德报应？在这样一个备受困扰的社会里，人民会对自己能否通过工作或者学习来改善自身的境遇产生怀疑。这种情况由于腐败而不负责任的司法制度而变得更加无法容忍，没有一个平民百姓会指望从这一制度中得到公平的补偿。在这样一个世界里，妖术既是一种权力的幻觉，又是对每个人的一种潜在的权力补偿。
